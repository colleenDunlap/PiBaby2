Raspberry Pi

As I mentioned before, this website is hosted on a Raspberry Pi. This means that all the HTML/CSS/JS code, as well as corresponding data and image files are being stored on a computer the size of a credit card, connected to my home ethernet (and TV, thank you roommates <3 ). If you are seeing this, that means the Raspberry Pi is connected to power, and all requests to this DNS address are being forwarded to the Pi's IP address.
